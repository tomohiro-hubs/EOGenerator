# 設備概要文ジェネレーター

太陽光発電所の設備概要文を定型フォーマットで自動生成する静的Webアプリケーションです。
入力された数値をもとにDC/AC容量を自動計算し、ワンクリックでコピー可能なテキストを生成します。

## 🚀 特徴

*   **完全クライアントサイド動作**: サーバー処理を行わず、ブラウザのみで完結します。
*   **リアルタイムプレビュー**: 入力と同時に生成結果を確認できます。
*   **自動計算**:
    *   DC容量 = (パネル容量 × 枚数) / 1000 （小数第2位固定）
    *   AC容量 = パワーコンディショナー容量 × 台数
*   **レスポンシブデザイン**: PC/スマホ両対応。
*   **設定変更が容易**: `data.json` を編集するだけで会社名やテンプレートを変更可能。

## 🛠 技術スタック

*   **HTML5 / CSS3**
*   **React** (via CDN)
*   **Tailwind CSS** (via CDN)
*   **Babel Standalone** (via CDN)
*   **React Router** (HashRouter)

ビルドステップ（`npm install` や `npm run build`）は**不要**です。`index.html` をブラウザで開くだけで動作します。

## 📖 使い方

1.  **サイト名**: 案件名や場所を入力（例：福岡県筑後市②）。
2.  **種類**: プルダウンから選択。リストにない場合は「カスタム」を選んで手入力。
3.  **太陽光パネル**: 1枚あたりのワット数(W)と枚数を入力。
4.  **PCS**: 1台あたりの出力(kW)と台数を入力。
5.  **コピー**: 右側（スマホは下部）の「テキストをコピーする」ボタンをクリック。

## ⚙️ 設定の変更 (data.json)

`public/data.json`（またはルートの `data.json`）を編集することで、マスターデータを変更できます。

```json
{
  "company": {
    "name": "会社名",
    "hq": "本社所在地",
    "contact": { ... }
  },
  "typePresets": [
    "選択肢1",
    "選択肢2"
  ],
  "templates": {
    "handover": "テンプレート文言..."
  }
}
```

### テンプレート変数

| 変数名 | 説明 |
| :--- | :--- |
| `{companyName}` | 会社名 |
| `{siteName}` | 入力されたサイト名 |
| `{typeText}` | 選択された種類（またはカスタム入力） |
| `{panelW}` | パネル容量(W) |
| `{panelCount}` | パネル枚数（カンマ区切り） |
| `{dcKw}` | 計算されたDC容量（kW、小数2位固定） |
| `{pcsKw}` | PCS容量(kW) |
| `{pcsCount}` | PCS台数（カンマ区切り） |
| `{acKw}` | 計算されたAC容量（kW） |

## 📦 デプロイ方法

このプロジェクトを公開するには、**Publishタブ**から公開ボタンを押してください。
GitHub Pagesなどでホスティングする場合も、すべてのファイルをそのままアップロードするだけで動作します。
